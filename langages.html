Saad : C'est quoi JAVA ?

C’est quoi ? Le Java est un langage de programmation orienté objet qui a été développé par Sun Microsystems dans les années 90. C’est l’un des langages de programmation les plus populaires, devenu un standard chez les programmes d’entreprise, dans les contenus et jeux Web ou encore les applications mobiles. C’est également le langage de programmation utilisé pour le système d’exploitation mobile Android. Le Java est fait pour travailler sur de nombreuses plateformes : un programme codé sur Mac OS X pourrait par exemple fonctionner sur Windows.
